# PaperNote 路线图

PaperNote 的目标是成为一款稳定、易用、可离线工作的 Windows 墨迹笔记应用。路线图会根据真实使用反馈调整，不代表时间承诺。

## 当前重点：v1.0.x 稳定性

- 收集不同 Windows 10/11 设备、触控笔和数位板的兼容性反馈。
- 改善首次启动、示例内容、空状态和安装说明。
- 继续降低长笔记、高清 PDF 和大量缩略图场景的内存占用。
- 修复数据保存、PDF 导入导出和窗口关闭方面的边界问题。
- 完善自动化回归测试和发布检查。

## 下一阶段：使用体验

- 更灵活的快捷键与工具栏设置。
- 更多内置纸张、封面和可共享模板。
- 更清晰的 PDF 批注工作流与导出预览。
- 更好的高分屏、不同窗口尺寸和多显示器适配。
- 可选的便携数据目录和更直观的备份管理。

## 长期候选

- 插件式导入导出能力。
- 更丰富的本地检索和内容组织方式。
- 在不牺牲本地优先原则的前提下探索可选的本地智能能力。

## 明确不在当前范围

- 账号、订阅和服务端依赖。
- 跨设备云同步。
- 多人实时协作和在线评论。
- 强制联网的 OCR、AI 或内容分析服务。

如果你希望某项能力优先实现，请在 GitHub Issues 中说明使用场景，而不仅是功能名称。
