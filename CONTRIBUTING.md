# 参与贡献

感谢你改进 PaperNote Desktop。

## 开始之前

1. 确认修改不包含第三方商业软件的代码、图标、截图、模板、字体或提取资源。
2. 不要提交私人笔记、真实姓名、账号信息、绝对用户路径、密钥或构建产物。
3. 新功能应继续符合本地优先、可离线使用和 Windows 桌面交互的产品范围。

## 本地验证

需要安装 .NET 9 SDK。在仓库根目录运行：

```powershell
.\scripts\test.ps1
```

脚本会完成 Release 构建、核心后台烟雾测试和隐藏 WPF 界面测试，不会操控鼠标。

## 提交建议

- 一次提交只处理一个明确问题；
- 为行为变化补充或更新后台测试；
- 更新 README 或产品方案中受影响的说明；
- 保持现有数据格式兼容，必要时提供迁移逻辑；
- UI 修改应保持 PaperNote 的独立视觉，不以一比一还原第三方产品为目标。

## Pull Request

请说明变更目的、主要行为、测试结果和可能的数据兼容影响。如果修改界面，建议附上由本项目生成的前后截图。
