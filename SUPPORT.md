# 获取帮助

- 使用问题和可复现的普通缺陷：提交公开 Issue，并注明 PaperNote 版本与运行平台。
- Windows 问题：请提供 Windows 版本、设备类型，以及触控笔或数位板型号（如适用）。
- Android 问题：请提供 Android 系统版本、设备型号、屏幕形态，以及触控笔型号（如适用）。
- 跨设备文件问题：请说明文件来自 Windows 还是 Android、经过的导入导出步骤，以及能否用新建的空白测试笔记复现。
- 功能建议：说明使用场景、预期行为和本地优先约束。
- 安全问题：按照 [SECURITY.md](SECURITY.md) 私下报告。
- 请勿上传包含真实笔记、个人文件、账号信息、系统绝对路径、设备标识、完整日志或整库备份的附件；请先制作不含私人数据的最小示例。
