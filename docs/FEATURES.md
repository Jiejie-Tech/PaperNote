# PaperNote 功能清单

本文档记录 PaperNote Desktop 当前已经实现的详细功能。面向普通使用者的简要介绍和下载方式请返回 [README](../README.md)。

> 当前数据格式版本：13
> 当前公开版本：v1.0.0

## 功能总览

### 书写与编辑

- 圆珠笔、钢笔、画笔、铅笔、荧光笔、橡皮、浏览和套索工具
- 压感曲线支持关闭、柔和、标准和强烈；笔迹平滑支持关闭、标准和强力
- 内置细黑圆珠笔、蓝色钢笔、红色批注笔和黄色荧光笔快捷预设
- 颜色、粗细、橡皮尺寸、缩放、全屏和只读模式
- 笔迹撤销、重做、清空、ISF 导入和 ISF 导出
- 区域混合套索可同时选中笔迹、文本、图片和形状
- 按住 `Ctrl` 可追加混合选区
- 自动保存，关闭窗口前会以非阻塞方式再次保存；连续点击关闭也不会重复进入保存流程

### 笔记本与页面

- 独立设计的本地资料库、封面、文件夹、分类和回收站
- 新建、打开、重命名、移动到回收站、恢复和永久删除笔记本
- 页面缩略图、添加、复制、删除、上移和下移
- 添加页面可选择插入到当前页前、当前页后或笔记末尾，并可直接选择空白、点阵、横线或方格纸
- `Ctrl+Enter` 沿用当前纸张，在当前页后快速插入一页
- 可把当前模板和颜色收藏为“常用纸张”，最多 8 个，并在当前页前、后或笔记末尾快速插入
- 每页可设置独立标题，并在侧栏直接添加或取消书签
- 页面搜索支持页码、页面标题和页面内文字对象
- “仅书签”筛选可快速定位重点页面
- 页面网格总览以大缩略图展示整本笔记，支持搜索、仅书签筛选和多选批量操作
- 页码输入框可直接跳转；底部按钮和 `PageUp` / `PageDown` 可快速前后翻页
- 底部提供页面访问后退/前进，可回到刚才浏览过的位置，并支持浏览器后退/前进硬件键
- 自动保存当前阅读页；再次打开笔记时恢复上次阅读位置
- 每页可设置一级、二级或三级目录级别；笔记目录支持搜索、缩进显示和双击跳转
- `Ctrl+J` 聚焦页码输入框，`Ctrl+Shift+O` 打开页面总览，`Ctrl+Shift+D` 打开笔记目录，`Esc` 关闭覆盖层
- 页面缩略图使用内存缓存，搜索、筛选和长笔记刷新时不再重复渲染未变化页面
- `Ctrl` / `Shift` 多选页面
- 多选页面稳定批量上移、下移、复制、删除、添加书签和取消书签
- 批量旋转 PDF 页面、批量裁边、清除裁剪和恢复原始状态
- 空白、点阵、横线和方格纸张；支持多种纸张颜色
- 可将 PNG、JPG、BMP、GIF 或 WebP 图片替换为当前页纸张，也可批量插入为独立图片页
- 共享模板库可保存当前页纸张，并跨笔记应用或插入；最多保留 24 个普通/图片纸张模板

### 页面对象

- 插入并编辑文本、图片、矩形、圆角矩形、椭圆、三角形、菱形、箭头和直线
- 对象移动、缩放、旋转、锁定、解锁和删除
- 对象复制、剪切、粘贴、组合、取消组合和图层调整
- 多对象对齐、等距分布、样式、透明度、文字大小和描边设置
- 文本、图片和形状对象可链接到本笔记任意页面；编辑模式下 `Ctrl+单击` 打开，只读模式下直接单击打开

### 多笔记切换

- 编辑器顶部显示真正可见的横向标签条，最多 8 本，可直接切换或单独关闭
- `Ctrl+Tab` / `Ctrl+Shift+Tab` 循环切换，`Ctrl+W` 关闭当前标签
- 返回书架后标签仍保留；删除或重命名笔记时同步更新标签
- 重新启动后恢复最近标签列表和书架排序设置

### PDF

- 导入多页 PDF，并转换为笔记本内部的 840×1188 PNG 背景
- 导入前可输入 `1-3,5,8-10` 等页码范围，支持全部、奇数页、偶数页快捷选择和最多 300 页限制
- 所选 PDF 页面可插入当前页前、当前页后或笔记末尾，只渲染实际选择的页面
- PDF 原文件移动或删除后，笔记仍能继续显示和批注
- 当前 PDF 页面支持左转、右转、5%/10% 裁边、清除裁剪和恢复原始状态
- 自定义裁剪编辑器支持四边独立比例、滑块精调和直接拖动裁剪框，取消时不会修改页面
- 导出整本笔记或多选页面为扁平 PDF
- 导出前可选择较小文件、标准或高清质量，以及 A4、A5 或 Letter 页面尺寸
- 导出内容包含纸张、模板、PDF 背景、笔迹、文本、图片和形状

### 历史版本与数据安全

- 覆盖保存前自动保护旧版本
- 自动版本最短间隔为 2 分钟，单本笔记最多保留 30 份
- 顶部“历史版本”可手动创建当前版本并恢复旧内容
- 恢复历史版本前会自动创建“恢复前保护点”
- 笔记和版本文件均采用临时文件写入后替换，避免半写入文件
- 永久删除笔记本时同步清理对应历史版本
- 可把整个资料库和全部历史版本导出为 `.papernote-library.zip`，并从备份包恢复
- 恢复前先校验包内所有笔记和历史版本；相同内容跳过，同名冲突作为副本导入，不直接覆盖

## 整库搜索与文字提取

- 书架支持搜索笔记本标题、文件夹、页面标题、可编辑文字对象和 PDF/图片来源文件名
- 多个关键词可跨字段组合匹配，并在卡片上显示命中位置
- 书架支持最近访问筛选，以及按最近修改、最近打开和标题排序
- 可复制当前页文字、复制整本笔记文字或导出整本为 TXT
- 当前文字提取只覆盖已有可编辑文本、页面链接说明和来源元数据；**不包含手写 OCR，也不把搜索描述为 OCR 搜索**

## 当前范围边界

PaperNote 当前专注于单机、本地优先的 Windows 笔记体验，以下能力不在现阶段范围内：

- 账号体系、登录和跨设备云同步
- 多人实时协作、共享链接和在线评论
- 服务端存储、在线 OCR、AI 摘要或联网内容处理
- 手写内容 OCR；目前搜索覆盖页面标题、可编辑文字对象、链接说明和来源元数据
