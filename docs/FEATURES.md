# PaperNote 功能清单

> 更新日期：2026-07-23。以下仅列出当前仓库已实现或明确限制的离线能力。

## 资料库与数据可靠性

- 本地笔记本、文件夹、标签、收藏、置顶、最近访问、搜索和回收站。
- 自动保存、原子替换、历史版本、恢复中心、只读损坏检查和另存抢救副本。
- 整库备份格式 3：包含笔记、历史版本和音频附件，并校验文件长度与 SHA-256。
- 单本笔记 PNENC2 密码保护；兼容旧 PNENC1，加密自动保存、历史版本和备份保持保护状态。
- Windows 与 Android 通过 `.papernote` 文件、导出文件或整库备份离线交换数据。

## 手写、画布与对象

- 钢笔、荧光笔、压力、倾角、平滑、透明度、颜色和粗细预设。
- 整笔橡皮、局部橡皮、撤销/重做、单指平移、双指缩放和视口裁剪绘制。
- 自由形状混合套索，可同时处理笔迹、文本、图片、形状和签名式对象。
- 对象移动、缩放、旋转、复制、删除、组合/取消组合、锁定、前移/后移和内部页面链接。
- 图层创建、重命名、显示/隐藏、锁定、排序和活动图层切换。
- 文本、图片、直线、箭头、矩形、圆形、三角形、菱形、星形和公式对象。

## 几何与课堂工具

- 直线与 45° 吸附、矩形/椭圆规整。
- 选区长度测量、PDF 测量比例校准。
- 指定长度/角度的标尺线、角度辅助线、平行线和垂线构造。
- 胶带工具、激光笔、演示模式、个人素材库和笔迹到录音的反向跳转。
- 混合选区导出为白底 PNG；Android 可通过系统分享保存或发送。

## 页面、模板与资源

- 页面新建、复制、删除、重命名、书签、移动到开头/末尾、多选和批量重排。
- 缩略图网格、快速跳转、访问历史、文档目录和多种纸张模板。
- 自定义图片纸张、常用模板收藏和跨笔记共享模板。
- 个人素材分类、搜索、收藏、插入和管理。
- 素材与模板资源包本地导入/导出，清单数量和 SHA-256 完整性校验，重复项跳过。

## PDF 学习工作流

- PDF 页码范围导入、进度、取消、失败续接、旧缓存清理和 200 MB/500 页边界。
- 原 PDF 保持只读；页面背景支持旋转、裁剪和高质量扁平导出。
- 离线提取 PDF 原文、字词坐标、目录/书签和内部 GoTo 链接。
- PDF 原文搜索、查看和复制；统一批注列表、类型/颜色筛选和页面文字评论。
- 页面范围导出 PDF，或提取为新的 `.papernote` 并重映射内部链接。
- 覆盖层文本填写框、复选框、必填校验、图片/选区签名和测量比例校准。
- 合并本地笔记本，重新生成页面、图层、笔迹、对象、组、评论、录音和目录 ID。
- Windows 双页阅读窗口、独立参考页窗口；Android 双页/参考页只读视图。

## 本地搜索、OCR 与公式

- 搜索笔记/页面标题、标签、文本对象、PDF 文本、评论、来源名称及 OCR/手写识别结果。
- 随安装包提供离线 OCR 模型；支持当前页与整本批量识别、进度、取消和校对。
- OCR 保存文本块区域、阅读顺序、置信度、更新时间与待校对状态。
- 套索手写转为可编辑文字。
- 基础规则把普通文字转换为 LaTeX；支持编辑、复制和导出 SVG。

## 本地录音

- 页面级录音、播放、暂停、停止、重命名和删除。
- 本地波形、开头/25%/50%/75% 跳转、播放笔迹高亮和选中笔迹反向跳转。
- 时间标记添加、移动、重命名和删除。
- 非破坏性播放裁剪范围；原音频文件保持不变。
- Android：省空间 32 kHz/64 kbps、标准 44.1 kHz/128 kbps、高质量 48 kHz/192 kbps。
- Windows：使用系统默认录音设备，保存后离线分析 WAV 波形。

## 平台和可访问性

### Windows

- WPF 桌面应用，适配鼠标、触控、数位板和 Windows Ink。
- 完整、专注书写、阅读批注、极简四种工具栏布局。
- 只读双页窗口和独立参考页窗口，参考阅读时主编辑窗口保持可编辑。
- 基础键盘焦点、自动化名称、高对比度资源和隐藏 UI 后台测试。

### Android

- .NET MAUI / Android 客户端，适配手机、平板、触控笔和触摸手势。
- 两排可滚动/响应式工具栏，关键按钮至少 48 dp；完整、专注书写、阅读批注、极简布局可保存恢复。
- 高级工具、PDF、页面、对象、图层和录音入口不会因窄屏永久隐藏。
- Android 7.0+；应用不声明网络权限。

## 安全本地扩展

- 从 ZIP 导入声明式扩展清单；支持 Uppercase、Prefix、Replace、RegexReplace 等受限文字命令。
- 不加载 DLL、不运行脚本、不访问网络；限制扩展数、命令数、输入输出长度和正则执行时间。
- 非法操作、非法正则、重复命令和损坏包会在导入阶段拒绝。

## 明确边界

- 公式转换是基础规则，不是成熟数学识别模型。
- PDF 表单、签名和测量是 PaperNote 覆盖层；签名不是证书型数字签名，也不写回 AcroForm。
- 双页/参考页为只读辅助阅读，不是同屏双画布编辑。
- Windows 暂无可靠麦克风设备枚举和三档录音质量。
- 已有基础无障碍支持，但未通过完整 WCAG/TalkBack/Narrator 专项认证。
- 不包含账号、云同步、联网 AI、遥测、广告、多人协作或在线商店。

## Implementation status — 2026-07-23

本节是当前离线版本能力边界的统一说明。

### 已实现并纳入仓库测试

- Windows 与 Android 共用 PaperInk、页面对象和格式 19 数据模型；支持压力、倾角、平滑、整笔/局部擦除、透明度、图层、对象组合、锁定、层级和自由形状混合套索。
- 几何工具覆盖直线与 45° 吸附、矩形/椭圆规整、长度测量、可设长度和角度的标尺线、角度辅助线、平行线及垂线构造。
- PDF 支持 200 MB/500 页边界、范围导入、进度/取消/失败续接、SHA-256 缓存、原文与字词坐标提取、搜索/复制、目录、内部链接、评论、批注筛选、页面旋转、范围导出与提取。
- 高级 PDF 覆盖层支持文本填写框、复选框、必填校验、图片/选区签名和测量比例校准；可合并本地笔记本并安全重映射页面、图层、笔迹、对象、链接、目录和录音时间点 ID。
- 本地识别支持当前页与整本批量 OCR、进度/取消、区域/置信度/校对状态保存、选区手写转文字；基础数学规则可生成并编辑 LaTeX，复制公式并导出 SVG。
- Windows 提供双页只读阅读窗口和独立参考页窗口；Android 提供双页/参考页只读视图，返回后继续编辑原页。两端均可保存工具栏布局，移动端关键按钮保持至少 48 dp 触控尺寸。
- 个人素材库和共享模板支持分类、搜索、收藏及本地资源包导入/导出；声明式本地扩展只允许受限文字操作，不加载 DLL、不运行脚本或第三方代码。
- 页面录音支持波形、播放/暂停、时间标记、笔迹双向跳转、标记移动/重命名/删除和非破坏性播放裁剪；Android 另提供省空间、标准、高质量三档录音质量。
- 保存采用临时文件验证后替换；恢复中心、只读损坏检查、历史版本、整库备份格式 3、PNENC2 笔记级密码保护和 PNENC1 兼容均已纳入测试。
- 当前 `.papernote` 内部格式为 `FormatVersion 19`：格式 17 引入录音波形，格式 18 引入批量 OCR 元数据，格式 19 引入 PDF 字词坐标/测量比例、表单/签名/公式元数据和录音质量/裁剪字段。
- 后台回归覆盖 Core、格式 19 往返、PDF/公式/合并/资源包/安全扩展、Windows 隐藏 UI、Android Release 编译与 APK 静态检查；整个测试过程不操控用户鼠标。

### 当前明确边界

- 数学识别是基础规则转换，不是成熟公式识别模型；复杂手写公式仍需手动编辑 LaTeX。
- PDF 表单、签名和测量以 PaperNote 页面覆盖层保存，不直接写回原 PDF 的 AcroForm；本地签名是图片签名，不是证书型数字签名。
- PDF 内部链接仅在目标页已导入当前笔记时可跳转；原文可查看和复制，但尚未提供逐字选择、原生文本高亮或直接修改原 PDF。
- Windows 录音使用系统默认录音设备，目前没有可靠的设备枚举和三档编码质量；Android 已提供三档质量。播放裁剪不会改写原音频文件。
- 双页与参考视图是只读辅助阅读：Windows 主窗口可继续编辑，Android 返回编辑器后继续编辑原页，不是同屏双画布编辑。
- 扩展系统只执行声明式文字命令；不会加载不受信任代码。无障碍提供基础语义、键盘焦点和高对比度资源，但未宣称通过完整 WCAG、TalkBack 或 Narrator 专项认证。
- 200 MB/500 页边界已自动化验证；低内存 Android、多品牌触控笔、超长录音和长时间运行仍需要持续真机压力测试。
- 账号、云同步、联网 AI、遥测、广告、多人协作和在线商店不在本地离线版本范围内。
- APK、ZIP、签名密钥、构建输出和私人笔记属于发布产物或本地数据，不提交到源码仓库。
