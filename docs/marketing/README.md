## 统一短介绍

PaperNote 是一款支持 Windows 与 Android 的本地优先手写笔记应用。它提供触控笔压力输入、钢笔和荧光笔、纸张模板、分页管理、PDF 导入批注、文本/图片/形状、搜索、回收站和整库备份。核心数据默认保存在设备本地，不要求账号，也不依赖内置云服务；可通过 `.papernote` 文件在电脑、手机和平板之间传递笔记。
