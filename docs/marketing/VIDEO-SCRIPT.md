# PaperNote 45 秒宣传视频脚本

画幅：1920×1080，16:9
时长：约 45 秒
形式：无配音、字幕驱动、轻量背景音乐可选
主色：深青绿 `#0F766E`、浅薄荷 `#DDF5F1`、纸白 `#F7FBFA`

## 0–4 秒：开场

画面：PaperNote 标志和名称从纸面网格中出现。
字幕：`PaperNote`
副标题：`本地优先的 Windows 开源墨迹笔记应用`

## 4–10 秒：资料库

画面：资料库截图进入，笔记卡片依次高亮。
字幕：`把课程、计划和灵感整理在本地资料库`

## 10–18 秒：自然书写

画面：编辑器截图放大，沿现有笔迹方向出现动态墨迹，工具栏笔型依次点亮。
字幕：`触控笔 · 数位板 · 鼠标`
补充：`多笔型、压感和平滑设置`

## 18–25 秒：纸张和分页

画面：空白、点阵、横线和方格纸标签轮换，左侧缩略图向下展开。
字幕：`模板、分页、书签和目录`

## 25–32 秒：PDF 批注

画面：PDF 页面卡片进入，叠加荧光标记、手写线条和裁剪框。
字幕：`导入 PDF，批注后重新导出`

## 32–38 秒：本地和安全

画面：电脑轮廓、文件和盾牌图标组合。
字幕：`无需账号 · 无需订阅 · 无需云服务`
补充：`自动保存、历史版本和整库备份`

## 38–45 秒：结尾行动

画面：产品封面、Windows 标志化文字和下载按钮。
字幕：`PaperNote Desktop v1.0.0`
补充：`Windows 10/11 · 免费开源`
按钮：`立即下载`
尾注：`GitHub / Gitee 搜索 PaperNote`

## 录屏完整版建议

如果后续补充真实操作录屏，按以下顺序录制：

1. 打开资料库；
2. 新建笔记；
3. 使用钢笔、荧光笔和橡皮；
4. 更换点阵和方格纸；
5. 添加页面、书签和目录；
6. 导入 PDF 并批注；
7. 页面总览和搜索；
8. 导出 PDF；
9. 返回资料库并执行备份。
