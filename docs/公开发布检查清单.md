# 公开发布检查清单

在推送 GitHub、Gitee 或制作 Release 前逐项确认：

- [ ] `scripts/test.ps1` 全部通过；
- [ ] `src/`、`tests/`、`docs/` 中没有来源混淆或比较式宣传表述；
- [ ] 没有未经授权的 Logo、图标、截图、模板、字体或提取资源；
- [ ] 没有私人笔记、真实个人信息、密钥、密码和绝对用户路径；
- [ ] `LICENSE` 与 `THIRD-PARTY-NOTICES.md` 已保留；
- [ ] 新增依赖已补充到第三方声明；
- [ ] 界面截图来自 PaperNote 自身；
- [ ] Windows 发布包由 `scripts/build-release.ps1` 生成；
- [ ] Android APK 与分发 ZIP 由 `scripts/build-android.ps1` 生成并通过签名验证；
- [ ] Android 签名文件和密码只保存在仓库外；
- [ ] Windows 与 Android 发布包均包含许可证、隐私说明和第三方许可；
- [ ] Android ZIP 内含 APK、APK 校验文件、安装指南和许可文件；
- [ ] Git 仓库没有提交 `dist/`、`artifacts/`、`work/` 或用户数据；
- [ ] 仓库名称、描述、Topics 和 Release 标题只描述 PaperNote 自身；
- [ ] Android 文档明确提醒卸载前备份应用私有数据；
- [ ] GitHub 与 Gitee 发布内容对应同一提交。

> Local-first handwriting and PDF annotation app for Windows and Android, built with .NET 10, WPF and .NET MAUI.
