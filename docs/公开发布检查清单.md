# 公开发布检查清单

在推送 GitHub、Gitee 或制作 Release 前逐项确认：

- [ ] `scripts/test.ps1` 全部通过；
- [ ] `src/`、`tests/`、`docs/` 中没有第三方产品复刻式表述；
- [ ] 没有第三方 Logo、图标、截图、模板、字体或提取资源；
- [ ] 没有私人笔记、真实个人信息、密钥和绝对用户路径；
- [ ] `LICENSE` 与 `THIRD-PARTY-NOTICES.md` 已保留；
- [ ] 新增依赖已补充到第三方声明；
- [ ] 界面截图来自 PaperNote 自身；
- [ ] 发布包由 `scripts/build-release.ps1` 生成；
- [ ] ZIP 内含项目许可证、隐私说明和第三方许可证；
- [ ] Git 仓库没有提交 `dist/`、`artifacts/`、`work/` 或用户数据；
- [ ] 仓库名称、描述、Topics 和 Release 标题不使用第三方商标引流；
- [ ] 项目对外描述明确为独立开源项目。

> Independent local-first handwriting and PDF annotation app for Windows, built with .NET 9 and WPF.
