# PaperNote Android 安装与使用

## 系统要求

- Android 7.0（API 24）或更高版本
- 建议预留至少 300 MB 可用空间，导入大型 PDF 时需要更多临时空间
- 支持手机和平板
- APK 包含 ARMv7、ARM64、x86 和 x64 ABI

## 获取与安装

正式发布页提供两种 Android 文件：

- 推荐下载：`PaperNote-Android-1.0.0.zip`，内含 APK、SHA-256 校验文件、安装说明和许可文件；
- 直接安装：`PaperNote-Android-1.0.0.apk`，适合已经明确需要单独 APK 的用户。

### 从压缩包安装

1. 下载并解压 `PaperNote-Android-1.0.0.zip`。
2. 在 Android 手机或平板上打开其中的 `PaperNote-Android-1.0.0.apk`。
3. 若系统阻止安装，只为当前文件管理器或浏览器临时开启“安装未知应用”。
4. 安装完成后关闭该来源的安装权限。
5. 可使用同目录的 `PaperNote-Android-1.0.0.apk.sha256` 校验 APK 完整性。

### 直接安装 APK

1. 下载 `PaperNote-Android-1.0.0.apk`。
2. 在文件管理器中打开 APK。
3. 若系统阻止安装，只为当前文件管理器或浏览器临时开启“安装未知应用”。
4. 安装完成后关闭该来源的安装权限。

首次启动后即可离线使用，无需注册。升级时请直接覆盖安装，不要先卸载旧版本。重要升级前仍建议先创建整库备份。

## 基本操作

### 资料库

- 新建、打开、搜索、排序、重命名和删除笔记。
- 回收站可恢复或永久删除笔记。
- “恢复中心”列出未完成草稿与损坏文件，可查看诊断并另存抢救副本；原文件不会被覆盖。

### 书写

- 手机默认开启手指书写；钢笔、荧光笔、整笔/局部橡皮、粗细、颜色、手指开关、撤销和重做固定在两排工具栏中。
- 触控笔支持压力与倾角；橡皮端自动切换擦除。高级工具栏可开启“几何”，自动规整直线、45°、矩形和椭圆。
- 双指缩放或移动时不会提交意外点迹。
- 大笔迹页面只绘制当前可见区域及少量预加载边界；橡皮先查询附近笔迹再做精确命中。

### 对象、页面与图层

- 手机使用底部页面入口，平板横向空间足够时显示侧栏。
- 可新增、复制、删除、重命名、移动和跳转页面。
- “PDF → 页面批量管理”可输入 `1-5,8,10-12` 这样的页码范围，批量复制、删除、移至首尾、旋转、书签、导出 PDF 或提取为新笔记本。
- 可插入文本、图片和常用形状。
- 自由形状混合套索可同时选择笔迹、文字、图片和形状，支持八类筛选、移动、缩放、旋转、复制、删除、组合、层级顺序、锁定、跨页复制/移动和批量样式；还可“导出选区为 PNG”，通过系统分享保存或发送。
- 图层支持新增、激活、显隐、锁定、透明度、重命名、合并和删除时迁移内容。

### PDF 学习、导入与批注

- 选择 PDF 后，应用先安全暂存文件并读取页数，再输入 `1-20,25,30-40` 这样的页码范围。
- 单个文件上限 200 MB，一次最多导入 500 页；导入过程中显示逐页进度。
- 可点击“取消导入”。已经完成的页面会保留在本地缓存，下次选择同一 PDF 和相同页码范围时自动续接。
- 页面按 840 × 1188 JPEG 背景写入笔记并保持原页比例；导入会占用笔记文件空间，请预留足够存储。
- 点击编辑器中的“PDF”可打开学习菜单：搜索 PDF 文本、查看书签与大纲、打开当前页内部链接、浏览批注列表、批量管理页面、范围导出和范围提取。
- 自带文本层的 PDF 会在本机离线提取文本；扫描版 PDF 没有文本层时搜索不到原文，但仍可正常书写批注。
- 原 PDF 目录/书签会被保存。内部链接只有目标页也已导入当前笔记时才能跳转，未导入目标会提示。
- 批注列表支持按类型筛选，可添加或删除当前页文字评论。

### 本地录音

- 当前页面可开始、暂停、继续和停止录音。
- 录音可播放、暂停、重命名、删除并添加命名时间标记。
- 录音期间书写会记录时间点并在本机采集麦克风振幅；音频使用 MPEG-4/AAC，保存在应用私有资料库并纳入整库备份。
- “查看波形并跳转”显示紧凑波形，可跳到开头、25%、50% 或 75%；播放时以黄色高亮对应笔迹，停止、切页或离开编辑器会自动清除。

## 文件交换

- 导出 .papernote 后，可通过聊天工具、数据线、网盘或局域网传给 Windows/其他 Android 设备。
- 导入 .papernote 会在本地资料库创建副本。
- PDF 可按页码范围异步导入为背景，查看进度、取消并续接，批注后导出全部/当前/指定页面为普通扁平 PDF。
- 指定页面可提取为新的 `.papernote` 并通过 Android 系统分享；PDF 文本、目录、内部链接和评论会随笔记文件传递。

PaperNote 不依赖网络，也不申请 INTERNET 权限；文件传输由用户选择的系统应用负责。

## 保存、恢复与后台行为

- 页面编辑会自动保存；切换笔记前保存任务会绑定正确文件。
- 应用进入后台或编辑页销毁时，会保存当前笔记并停止/收尾活动录音。
- 启动时会恢复较新的可读临时草稿。
- 恢复中心无法结构化读取文件时，仅显示原始片段，不会覆盖或删除原文件。

## 备份与卸载

Android 将资料库存放在应用私有目录。卸载、清除应用数据、恢复出厂设置或部分清理工具可能删除全部本地笔记。

操作前：

1. 打开“备份”；
2. 创建整库备份；
3. 使用系统文件选择器把 .pnbak 保存到下载目录、移动存储或另一台设备；
4. 最好执行一次恢复验证或至少核对 SHA-256；
5. 升级时覆盖安装，不要先卸载。

## 返回键行为

- 编辑器、搜索、设置、备份、恢复中心和回收站：返回资料库或上一页。
- 资料库根页：把应用移入后台，不触发异常退出。

## 故障排查

### 无法安装

确认 Android 版本符合要求、APK 完整，并只为当前文件来源临时开启“安装未知应用”。

### PDF 工具栏或学习菜单看不到

确认已进入编辑器，顶部两排工具栏应完整显示；“PDF”按钮位于编辑工具区。若屏幕字体或显示缩放很大，可临时调回系统默认后重启应用，并保存截图和设备型号反馈。

### 导入后没有出现

清除搜索条件并刷新资料库；损坏文件可在恢复中心查看。

### 大型 PDF 导入较慢

导入会逐页转换并显示进度。若处理时间较长，可点击“取消导入”；再次选择同一 PDF 和相同页码范围会复用已完成页面。若缓存或笔记体积较大，请释放存储空间后重试。

### 大笔迹页面仍有延迟

关闭省电模式和高负载后台应用。视口按需渲染能减少无关笔迹开销，但低端设备上的超长单页仍建议拆分。

## 高级离线工具与窄屏布局

- 编辑器顶部/更多菜单中的“高级离线工具”包含 PDF 原文、表单/签名、测量、公式、笔记合并、资源包、安全扩展和录音精细编辑。
- “工具栏布局”可选完整、专注书写、阅读批注或极简；选择会保存在本机，下次打开恢复。
- 手机窄屏不会永久隐藏钢笔类型、粗细、颜色、橡皮、手指书写、撤销/重做和高级工具；工具栏可按布局重新显示全部入口，关键按钮至少 48 dp。
- 双页/参考页是只读查看，返回后继续编辑原页。它们不会同时改动两页，避免手机误触。
- Android 录音质量可选：省空间 32 kHz/64 kbps、标准 44.1 kHz/128 kbps、高质量 48 kHz/192 kbps。
- 整本 OCR 运行时可取消；结果会保存在笔记中并进入离线搜索。首次使用模型来自安装包，不需要网络下载。

## Implementation status — 2026-07-23

本节是当前离线版本能力边界的统一说明。

### 已实现并纳入仓库测试

- Windows 与 Android 共用 PaperInk、页面对象和格式 19 数据模型；支持压力、倾角、平滑、整笔/局部擦除、透明度、图层、对象组合、锁定、层级和自由形状混合套索。
- 几何工具覆盖直线与 45° 吸附、矩形/椭圆规整、长度测量、可设长度和角度的标尺线、角度辅助线、平行线及垂线构造。
- PDF 支持 200 MB/500 页边界、范围导入、进度/取消/失败续接、SHA-256 缓存、原文与字词坐标提取、搜索/复制、目录、内部链接、评论、批注筛选、页面旋转、范围导出与提取。
- 高级 PDF 覆盖层支持文本填写框、复选框、必填校验、图片/选区签名和测量比例校准；可合并本地笔记本并安全重映射页面、图层、笔迹、对象、链接、目录和录音时间点 ID。
- 本地识别支持当前页与整本批量 OCR、进度/取消、区域/置信度/校对状态保存、选区手写转文字；基础数学规则可生成并编辑 LaTeX，复制公式并导出 SVG。
- Windows 提供双页只读阅读窗口和独立参考页窗口；Android 提供双页/参考页只读视图，返回后继续编辑原页。两端均可保存工具栏布局，移动端关键按钮保持至少 48 dp 触控尺寸。
- 个人素材库和共享模板支持分类、搜索、收藏及本地资源包导入/导出；声明式本地扩展只允许受限文字操作，不加载 DLL、不运行脚本或第三方代码。
- 页面录音支持波形、播放/暂停、时间标记、笔迹双向跳转、标记移动/重命名/删除和非破坏性播放裁剪；Android 另提供省空间、标准、高质量三档录音质量。
- 保存采用临时文件验证后替换；恢复中心、只读损坏检查、历史版本、整库备份格式 3、PNENC2 笔记级密码保护和 PNENC1 兼容均已纳入测试。
- 当前 `.papernote` 内部格式为 `FormatVersion 19`：格式 17 引入录音波形，格式 18 引入批量 OCR 元数据，格式 19 引入 PDF 字词坐标/测量比例、表单/签名/公式元数据和录音质量/裁剪字段。
- 后台回归覆盖 Core、格式 19 往返、PDF/公式/合并/资源包/安全扩展、Windows 隐藏 UI、Android Release 编译与 APK 静态检查；整个测试过程不操控用户鼠标。

### 当前明确边界

- 数学识别是基础规则转换，不是成熟公式识别模型；复杂手写公式仍需手动编辑 LaTeX。
- PDF 表单、签名和测量以 PaperNote 页面覆盖层保存，不直接写回原 PDF 的 AcroForm；本地签名是图片签名，不是证书型数字签名。
- PDF 内部链接仅在目标页已导入当前笔记时可跳转；原文可查看和复制，但尚未提供逐字选择、原生文本高亮或直接修改原 PDF。
- Windows 录音使用系统默认录音设备，目前没有可靠的设备枚举和三档编码质量；Android 已提供三档质量。播放裁剪不会改写原音频文件。
- 双页与参考视图是只读辅助阅读：Windows 主窗口可继续编辑，Android 返回编辑器后继续编辑原页，不是同屏双画布编辑。
- 扩展系统只执行声明式文字命令；不会加载不受信任代码。无障碍提供基础语义、键盘焦点和高对比度资源，但未宣称通过完整 WCAG、TalkBack 或 Narrator 专项认证。
- 200 MB/500 页边界已自动化验证；低内存 Android、多品牌触控笔、超长录音和长时间运行仍需要持续真机压力测试。
- 账号、云同步、联网 AI、遥测、广告、多人协作和在线商店不在本地离线版本范围内。
- APK、ZIP、签名密钥、构建输出和私人笔记属于发布产物或本地数据，不提交到源码仓库。

## Android 本地密码保护

1. 打开笔记本，点“更多”→“启用密码保护”。
2. 输入两次至少 8 个字符的密码。密码不会保存在手机设置中。
3. 退出笔记本后，书架显示锁标记；再次打开时需要解锁。
4. “更多”→“管理密码保护”可以更改密码或关闭保护。
5. 导入加密的 `.papernote` 时会先要求密码；密码错误不会覆盖或删除原文件。
6. 自动保存、回收站和整库备份按原加密状态工作，Windows 可直接打开同一文件。

> 忘记密码无法找回。加密用于保护静态文件，不承诺抵御已解锁设备上的恶意软件、屏幕录制或内存攻击。

## 个人素材库

1. 用“套索”圈选笔迹、文字、图片或形状。
2. 点“更多”→“保存到个人素材库”，输入名称。
3. 需要复用时点“更多”→“课堂与复习工具”→“我的素材”，选择素材后点“插入当前页”。
4. 在同一菜单中可删除不再需要的素材；删除素材不会影响已经插入笔记的内容。

素材只保存在当前 Android 设备的应用私有目录，最多 48 项。它不会自动随单个 `.papernote` 文件迁移，也不受笔记密码保护；但插入页面后的内容会成为普通笔迹/对象，并随该笔记保存、加密、备份和跨平台传递。卸载应用前如需保留素材，目前应先把重要素材插入一个专用笔记并导出该笔记。
## 课堂工具

Android 编辑器的“更多→课堂与复习工具”提供胶带、常用元素、个人素材、激光笔和演示模式。激光轨迹只在当前界面显示，不保存到文件。
