# PaperNote 快速使用指南

## 1. 下载和启动

1. 前往 GitHub Releases 下载 `PaperNote-Desktop-1.0.0-win-x64.zip`。
2. 将压缩包完整解压到一个普通文件夹，不要直接在压缩包中运行。
3. 双击 `PaperNote.Desktop.exe`。
4. 如果 Windows 显示来源提示，请确认文件来自本项目官方 Release，并核对 SHA256 后再继续。

系统要求：Windows 10 或 Windows 11，64 位系统。

## 2. 创建第一本笔记

在资料库右上角选择“新建笔记”，输入名称、选择文件夹和封面后即可进入编辑器。笔记默认保存在当前用户的“文档\PaperNote”目录。

## 3. 书写和编辑

- 在顶部工具栏选择圆珠笔、钢笔、画笔、铅笔或荧光笔。
- 调整颜色、粗细、压感和平滑级别。
- 橡皮用于擦除笔迹；套索可以选择笔迹、文字、图片和形状。
- 使用撤销和重做恢复最近的操作。

## 4. 管理页面

- 左侧缩略图栏用于切换、添加、复制、删除和移动页面。
- 可以为页面设置标题、书签和目录级别。
- `Ctrl+Enter` 会沿用当前纸张并在当前页后插入新页。
- `Ctrl+Shift+O` 打开页面总览，`Ctrl+Shift+D` 打开目录。
- `Ctrl+J` 快速跳转到指定页码。

## 5. 导入并批注 PDF

选择顶部“导入 PDF”，设置页码范围和插入位置。导入后可以像普通页面一样书写、添加文字和形状，并对页面旋转或裁边。完成后使用“导出 PDF”生成带批注的新文件。

## 6. 保存、历史版本和备份

- PaperNote 会自动保存，仍建议在重要操作后手动保存。
- “历史版本”可以创建保护点并恢复旧内容。
- 资料库中的备份功能可以导出 `.papernote-library.zip`，用于迁移或恢复完整资料库。
- 请定期把备份包复制到另一块磁盘或你自行选择的安全位置。

## 7. 数据位置

```text
文档\PaperNote\Notebooks   正式笔记本
文档\PaperNote\Backups     自动和手动历史版本
本地应用数据\PaperNote      标签、排序和共享模板设置
```

PaperNote 默认不要求账号，也不会主动上传笔记内容。

## 8. 常见问题

### 双击后没有启动

请先完整解压，确认使用 Windows 10/11 64 位系统，并尝试从解压后的文件夹再次运行。

### Windows 显示安全提醒

v1.0.0 暂未使用商业代码签名证书。请只从官方 GitHub/Gitee 发行页下载，并核对发布页提供的 SHA256。

### 可以同步到其他电脑吗

PaperNote 当前没有内置云同步。可以通过整库备份功能手动迁移，也可以自行把备份文件存放到你信任的同步工具中。

### 支持手写 OCR 吗

暂不支持手写 OCR。当前搜索覆盖页面标题、可编辑文字对象和部分来源元数据。

### 如何报告问题

请在 GitHub Issues 中说明 Windows 版本、设备类型、操作步骤、预期结果和实际结果；如果包含私人笔记，请先删除或遮挡敏感内容。
