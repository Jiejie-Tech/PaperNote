# PaperNote 用户指南

PaperNote 是一款本地优先的手写笔记与 PDF 批注应用。Windows 与 Android 可以通过 .papernote 文件或整库备份交换数据。

## 1. 资料库

- 新建、打开、重命名和删除笔记；删除后先进入回收站。
- 使用标题、页面标题、标签、文本对象、已存识别文本或来源名称搜索。
- 使用文件夹、排序和最近打开时间整理长期资料。
- 发现未完成草稿或损坏文件时，进入“恢复中心”查看诊断。

## 2. 书写与导航

### Windows

- 使用触控笔、鼠标或触控输入书写。
- 选择钢笔、荧光笔、整笔/局部橡皮，并调整颜色、粗细和压感相关设置。
- 使用撤销、重做、缩放、平移和页面侧栏。

### Android

- 手机端默认开启手指书写；触控笔支持压力、倾角和橡皮端。
- 两排工具栏固定显示笔型、粗细、颜色、手指开关、撤销和重做。
- “手指：开”时单指书写；关闭后单指平移；双指始终可缩放和平移。
- 大笔迹页面按当前视口绘制，移动页面时会预加载边缘内容。
- 系统返回键从编辑器或设置返回资料库；资料库根页返回会把应用移入后台。

## 3. 页面、对象与图层

- 新增、复制、删除、重命名、移动页面，并通过缩略图、页码、搜索或大纲跳转。
- 插入文本、图片、直线、箭头、矩形、圆形、三角形、菱形、星形等对象。
- 使用自由形状套索圈选笔迹、文字、图片或形状；可切换八类筛选，并对混合选区移动、缩放、旋转、复制、删除、跨页复制/移动或批量修改颜色、粗细、透明度和笔型。
- 图层可新增、重命名、显隐、锁定、调透明度、合并和删除；删除图层时内容会迁移到安全图层。

## 4. PDF 工作流

1. 选择“导入 PDF”。
2. 选择页码范围和插入位置。
3. 在背景页上书写或添加文本、图片和形状；需要时旋转或裁剪背景。
4. 选择“导出 PDF”生成包含批注的扁平 PDF。

导出不会修改原 PDF。大型 PDF 目前仍需等待逐页处理，尚无导入进度和取消按钮。

## 5. 本地录音与时间标记

- 在当前页面开始录音；可以暂停、继续和停止。
- 录音可播放、暂停、重命名或删除。
- 可手动添加命名时间标记；录音期间书写时会记录相应时间点。
- 录音文件只存放在本地资料库，并随整库备份导出。
- 当前没有波形视图，也不会在播放时自动高亮当时的笔迹。

## 6. 保存、恢复与数据自救

- 编辑过程自动保存，也可手动保存。
- 保存先写入临时文件并验证，成功后才替换正式文件。
- 应用启动时会自动恢复比正式文件更新且可读的临时草稿。
- “恢复中心”列出未完成草稿和损坏笔记，可查看标题、可读页数、诊断和原始片段。
- 选择“另存抢救副本”会创建新的笔记 ID 和文件；原损坏文件始终保留。
- 完全不可解析的文件只能查看原始片段，不能伪造为正常笔记。

## 7. 导入、导出和整库备份

- **导出笔记**：生成单个 .papernote，适合发送给另一台设备。
- **导入笔记**：选择 .papernote，在本地创建副本。
- **整库备份**：包含笔记、历史版本和音频附件，并记录长度和 SHA-256。
- **恢复备份**：导入前检查重复/越界路径、大小和哈希；校验失败不会写入资料库。

## 8. Android 数据提醒

Android 笔记位于应用私有目录。卸载、清除应用数据、恢复出厂设置或部分清理工具都可能删除内容。操作前务必把整库备份复制到应用目录之外；升级时应覆盖安装，不要先卸载旧版。

## 9. 常见问题

### 手指画不出线

确认已选择钢笔或荧光笔，并检查工具栏是否显示“手指：开”。

### 触控笔没有压力变化

部分设备不会提供压力数据，PaperNote 会使用普通线宽；这不影响笔迹保存。

### Windows 与 Android 显示略有差异

字体、PDF 渲染和平台绘制器可能带来轻微差异，但 PaperInk、对象、图层和录音元数据会保留。

### 书写很多后变慢

新版 Android 已按视口绘制并使用空间索引。若仍卡顿，先关闭省电模式、减少后台负载，并把极长内容拆分到多个页面；反馈问题时附设备型号、Android 版本和大致笔迹量。

## Implementation status — 2026-07-22

本节是当前离线版本能力边界的统一说明。

### 已实现并纳入仓库测试

- Windows 与 Android 共用 PaperInk；支持压力、倾角、平滑、整笔/局部擦除、透明度和图层归属。
- Windows 与 Android 均支持自由形状混合套索，可按全部内容、笔迹、钢笔、荧光笔、对象、文字、图片或形状筛选，并统一移动、缩放、旋转、复制、删除、跨页复制/移动及批量修改颜色、粗细、透明度和笔型。
- 页面图层支持新增、激活、显隐、锁定、透明度、重命名、合并和删除时迁移内容；隐藏内容不会从文件中丢失。
- 文本、图片和形状对象的旋转、透明度、锁定、隐藏、组合和图层字段可跨平台保存。
- 离线搜索覆盖笔记/页面标题、标签、文本对象、已存 OCR 文本、已存手写识别文本和来源名称。
- 保存采用“临时文件写入并验证后替换”；启动时可恢复较新的临时草稿。Windows 与 Android 都提供恢复中心，可只读检查损坏文件并另存为抢救副本，原文件保持不变。
- 读取时会修复空或重复 ID、无效图层引用、非有限墨迹数值及异常录音时间数据。
- 大墨迹页面使用空间索引；Android 按可见视口绘制并用局部候选执行橡皮命中，避免每帧扫描全部笔迹。
- 整库备份格式 3 包含笔记、历史版本和音频附件，记录长度与 SHA-256，并在导入前检查重复路径、越界路径、大小和内容完整性。
- Windows 与 Android 均支持页面级本地录音、播放/暂停、重命名、删除、命名时间标记和书写时自动标记。Windows 使用 WAV，Android 使用 MPEG-4/AAC。
- 压力测试覆盖 10,000+ 笔迹空间查询，以及 60 页、4,800 笔迹、600 对象、录音标记和图层关系的重复保存往返。

### 当前明确边界

- OCR 和手写识别结果可以保存并搜索，但仓库尚未内置真正的离线 OCR、手写转文字或数学识别引擎。
- 已提供自由形状混合套索、八类筛选和大批量笔迹样式修改；尚未提供几何吸附、自动形状识别和标尺。
- 录音暂不含波形视图、播放时笔迹高亮、麦克风设备选择和压缩质量控制。
- PDF 尚不含大型文档页面缓存、导入进度/取消、表单编辑、测量和文档内文字搜索。
- 完整屏幕阅读器语义、高对比度专项适配、加密设置界面和本地插件机制仍待完善。
- 账号、云同步、联网 AI、遥测、广告和多人协作不在离线版本范围内。
- APK、ZIP、签名密钥、构建输出和私人笔记属于发布产物或本地数据，不提交到源码仓库。
