# PaperNote 用户指南

PaperNote 是一款本地优先的手写笔记与 PDF 批注应用。Windows 与 Android 可以通过 .papernote 文件或整库备份交换数据。

## 1. 资料库

- 新建、打开、重命名和删除笔记；删除后先进入回收站。
- 使用标题、页面标题、标签、文本对象、PDF 文本、文字评论、已存识别文本或来源名称搜索。
- 使用文件夹、排序和最近打开时间整理长期资料。
- 发现未完成草稿或损坏文件时，进入“恢复中心”查看诊断。

## 2. 书写与导航

### Windows

- 使用触控笔、鼠标或触控输入书写。
- 选择钢笔、荧光笔、整笔/局部橡皮，并调整颜色、粗细和压感相关设置。
- 使用撤销、重做、缩放、平移和页面侧栏。

### Android

- 手机端默认开启手指书写；触控笔支持压力、倾角和橡皮端。
- 两排工具栏固定显示笔型、粗细、颜色、手指开关、撤销和重做。
- “手指：开”时单指书写；关闭后单指平移；双指始终可缩放和平移。
- 大笔迹页面按当前视口绘制，移动页面时会预加载边缘内容。
- 系统返回键从编辑器或设置返回资料库；资料库根页返回会把应用移入后台。

### 几何规整

开启几何辅助后，画完接近直线、矩形或椭圆的笔迹会在抬笔时规整；接近水平、垂直或 45° 的直线会自动吸附。Windows 在笔设置中开启，Android 在高级工具栏点击“几何：开/关”。普通手写只有满足识别阈值时才会被替换。

## 3. 页面、对象与图层

- 新增、复制、删除、重命名、移动页面，并通过缩略图、页码、搜索或大纲跳转。
- 页面批量管理支持多页复制、删除、移至开头/末尾、书签、PDF 背景左转/右转、范围导出和范围提取。Windows 从页面多选菜单进入；Android 从“PDF → 页面批量管理”进入并输入页码范围。
- 插入文本、图片、直线、箭头、矩形、圆形、三角形、菱形、星形等对象。
- 使用自由形状套索圈选笔迹、文字、图片或形状；可切换八类筛选，并对混合选区移动、缩放、旋转、复制、删除、跨页复制/移动或批量修改颜色、粗细、透明度和笔型。选区菜单还可裁剪导出白底 PNG；Android 会打开系统分享面板。
- 图层可新增、重命名、显隐、锁定、调透明度、合并和删除；删除图层时内容会迁移到安全图层。

## 4. PDF 学习工作流

### 导入和续接

1. 选择“导入 PDF”。
2. 输入页码范围，例如 `1-20,25,30-40`。Windows 还可选择插入位置。
3. 查看逐页进度；需要停止时点击“取消导入”。
4. 再次选择同一 PDF 和相同页码范围，会复用已完成页面继续导入。
5. 在背景页上书写、荧光标记或添加文本、图片和形状；需要时旋转或裁剪背景。

单个 PDF 上限 200 MB，一次最多导入 500 页。页面以 840 × 1188 JPEG 背景内嵌到笔记，因此大型讲义会增大 `.papernote` 文件。缓存默认最多使用约 1 GB，并优先清理 30 天以上旧任务。

### 搜索、目录和内部链接

- **Windows**：点击顶部“PDF 学习”，在同一面板中使用文本搜索、书签/目录、当前页内部链接和批注列表。
- **Android**：点击“PDF”，选择“搜索 PDF 文本”“书签与大纲”“当前页内部链接”或“批注列表”。
- PDF 自带文本层时，PaperNote 会在本机提取并保存文本；扫描版 PDF 没有文本层时不会自动识别，需要后续离线 OCR 能力。
- 原 PDF 目录/书签会与手工大纲一起显示。内部链接只有目标页也已导入当前笔记时才能跳转；否则应用会提示目标未导入。

### 批注和评论

- 钢笔、荧光笔、文字、图片和形状会作为 PaperNote 独立批注保存，不会写回原 PDF。
- 批注列表可按类型筛选；Windows 还可组合颜色筛选。
- 可在当前页面添加文字评论，并从批注列表定位或删除。

### 导出和提取

- 可导出全部页面、当前页或指定页码范围为普通扁平 PDF。
- 可把指定页面提取为新的 `.papernote`；新文件会生成独立 ID，并正确重映射所选页面之间的内部链接。
- 导出和提取都不会修改原 PDF。

## 5. 本地录音与时间标记

- 在当前页面开始录音；可以暂停、继续和停止。
- 录音可播放、暂停、重命名或删除。
- 可手动添加命名时间标记；录音期间书写时会记录相应时间点。
- 录音文件只存放在本地资料库，并随整库备份导出；`.papernote` 只保存相对路径、元数据、时间标记和波形峰值，不嵌入音频本体。
- 打开“波形与跳转/查看波形并跳转”可查看本地波形并跳到开头、25%、50% 或 75%。播放经过书写时间标记时，对应笔迹会以黄色高亮；停止、切页或离开页面后自动清除。

## 6. 保存、恢复与数据自救

- 编辑过程自动保存，也可手动保存。
- 保存先写入临时文件并验证，成功后才替换正式文件。
- 应用启动时会自动恢复比正式文件更新且可读的临时草稿。
- “恢复中心”列出未完成草稿和损坏笔记，可查看标题、可读页数、诊断和原始片段。
- 选择“另存抢救副本”会创建新的笔记 ID 和文件；原损坏文件始终保留。
- 完全不可解析的文件只能查看原始片段，不能伪造为正常笔记。

## 7. 导入、导出和整库备份

- **导出笔记**：生成单个 .papernote，适合发送给另一台设备。
- **提取页面**：按页码范围生成新的 .papernote，只包含所选页面和有效导航。
- **导入笔记**：选择 .papernote，在本地创建副本。
- **整库备份**：包含笔记、历史版本和音频附件，并记录长度和 SHA-256。
- **恢复备份**：导入前检查重复/越界路径、大小和哈希；校验失败不会写入资料库。

## 8. Android 数据提醒

Android 笔记位于应用私有目录。卸载、清除应用数据、恢复出厂设置或部分清理工具都可能删除内容。操作前务必把整库备份复制到应用目录之外；升级时应覆盖安装，不要先卸载旧版。

## 9. 常见问题

### 手指画不出线

确认已选择钢笔或荧光笔，并检查工具栏是否显示“手指：开”。

### 触控笔没有压力变化

部分设备不会提供压力数据，PaperNote 会使用普通线宽；这不影响笔迹保存。

### Windows 与 Android 显示略有差异

字体、PDF 渲染和平台绘制器可能带来轻微差异，但 PaperInk、对象、图层和录音元数据会保留。

### 书写很多后变慢

新版 Android 已按视口绘制并使用空间索引。若仍卡顿，先关闭省电模式、减少后台负载，并把极长内容拆分到多个页面；反馈问题时附设备型号、Android 版本和大致笔迹量。

## Implementation status — 2026-07-23

本节是当前离线版本能力边界的统一说明。

### 已实现并纳入仓库测试

- Windows 与 Android 共用 PaperInk 和页面对象模型；支持压力、倾角、平滑、整笔/局部擦除、透明度、图层及自由形状混合套索。
- 页面支持复制、删除、移动到开头/末尾、书签、PDF 背景旋转，以及按页码范围导出 PDF 或提取为新的 `.papernote`；提取时会生成新 ID 并重映射保留下来的内部链接。
- Windows 与 Android 的 PDF 导入支持 200 MB/500 页边界、逐页进度、取消、完整 SHA-256 内容指纹缓存、失败续接和旧缓存清理；原 PDF 始终不被修改。
- 对带文本层的 PDF，会在本机离线提取并保存页面文本、原 PDF 目录/书签和内部 GoTo 链接；Windows 与 Android 均可搜索文本、浏览目录并跳转到已导入的目标页。
- PDF 学习工作流包含统一批注列表、类型/颜色筛选和页面文字评论；评论、PDF 文本、目录和链接都保存在 `.papernote` 中，可随 Windows/Android 文件往返。
- 离线搜索覆盖笔记/页面标题、标签、文本对象、PDF 文本、文字评论、来源名称以及已保存的 OCR/手写识别结果。
- 保存采用“临时文件写入并验证后替换”；Windows 与 Android 均提供恢复中心、只读损坏检查和另存抢救副本。
- 整库备份格式 3 包含笔记、历史版本和音频附件，并校验长度与 SHA-256；页面级本地录音支持播放、暂停、重命名、删除、时间标记、本地波形、分段跳转、播放笔迹高亮和选中笔迹反向跳转。
- Windows 与 Android 均提供本机个人素材库，可把混合选区保存为最多 48 项可复用素材；重复插入会生成新内容 ID、重映射对象组并自动缩放。
- 当前 `.papernote` 数据格式版本为 17；加载器会修复常见无效 ID、坐标、图层引用、链接、目录、评论和波形数据。
- 后台回归覆盖 Core、真实 PDF 内容提取、格式 17 存储往返、几何规整、混合选区 PNG、录音波形、播放笔迹高亮、Windows 隐藏 UI、Android Release/AOT 与 APK 静态检查；测试过程不操控用户鼠标。

### 当前明确边界

- PDF 文本搜索依赖原文件自带文本层；扫描版 PDF 和图片尚未内置离线 OCR，因此只能作为页面图像批注。
- PDF 内部链接只有目标页也已导入到当前笔记时才能跳转；未导入目标会明确提示。
- 当前可搜索 PDF 文本，但尚未提供逐字选择、复制原文或直接修改原 PDF 的“原生文本高亮”；高亮仍以 PaperNote 独立批注保存。
- 尚未包含真正的离线 OCR、手写转文字、数学/LaTeX 识别、完整标尺和更高级的几何构造工具。
- 录音已包含本地波形、分段跳转、播放笔迹高亮和选中笔迹反向跳转；麦克风设备选择、质量设置及更精细的时间轴编辑仍待完善。
- PDF 可编辑表单、数字签名、测量、合并多个源 PDF、双页阅读和分屏仍未实现；200 MB/500 页边界仍需更多低内存 Android 真机长期压力验证。
- 完整屏幕阅读器语义、高对比度专项适配和本地插件机制仍待完善；笔记级本地密码保护已完成。
- 账号、云同步、联网 AI、遥测、广告和多人协作不在离线版本范围内。
- APK、ZIP、签名密钥、构建输出和私人笔记属于发布产物或本地数据，不提交到源码仓库。

## 给笔记本加密码

### Windows

打开笔记本后，在顶部点“密码保护”。首次启用需要确认风险并输入两次密码；已经保护时，该按钮显示“🔒 已保护”，再次点击可更改密码或关闭保护。

### Android

打开笔记本后点“更多”→“启用密码保护”；已经保护时入口会变为“管理密码保护”。

### 重要说明

- 密码至少 8 个字符，PaperNote 不保存，也不能找回。
- 密码仅在当前应用运行期间保留，关闭应用后需要重新输入。
- 书架仍会显示标题、封面、文件夹和页数；实际页面、笔迹、PDF 和录音信息均为密文。
- 恢复旧历史版本时，如果该版本使用旧密码，程序会单独询问旧密码，并继续用当前密码保护恢复后的内容。
- 关闭保护会把文件改回普通本地 `.papernote`，能访问文件的人可直接读取。

## 课堂工具

- Windows：在编辑器工具栏选择“课堂”，可添加胶带、常用元素、激光笔或进入演示模式。
- Android：在“更多”中选择“课堂与复习工具”。
- 胶带和元素可用选择工具移动、缩放、旋转、改色或锁定。
- 录音期间书写后，选中笔迹并选择“跳到关联录音”。
### 保存和复用个人素材

- **Windows 保存**：用套索选中内容，打开“选区”→“保存到个人素材库…”，输入名称。
- **Windows 插入/删除**：点顶部“课堂”→“我的素材”；选择一项即可插入，或从“删除素材”中移除。
- **Android 保存**：套索选中内容，点“更多”→“保存到个人素材库”。
- **Android 插入/删除**：点“更多”→“课堂与复习工具”→“我的素材”，选择素材后插入或删除。
- 素材可包含手写、文字、图片、形状和常用元素，插入后仍可移动、缩放、旋转、改色或擦除。
- 素材库只保存在当前设备，最多 48 项，不自动随 `.papernote` 迁移，也不受笔记密码保护。插入笔记后的内容则会随笔记保存、加密和备份。
