# 隐私说明

最后更新：2026-07-20

PaperNote Desktop 是本地优先的 Windows 笔记应用。

## 默认行为

- 应用不要求注册账号。
- 应用不包含广告、分析统计或遥测 SDK。
- 应用不会主动把笔记、笔迹、图片、PDF、搜索内容或使用记录上传到开发者服务器。
- 笔记和设置默认保存在用户自己的 Windows 文档目录中。

## 本地数据

应用可能在本机保存笔记文件、页面背景、缩略图、自动备份、工作区状态、纸张模板和导出的文件。这些内容由使用者自行管理和备份。把笔记导出、复制到同步盘或发送给他人时，相关平台的隐私规则由对应平台决定。

## 权限

应用仅在用户主动选择导入、导出或恢复备份时访问对应文件。PDF 转换和文字提取在本机执行。

## 数据安全

应用提供自动保存、历史版本和资料库备份，但这不能替代系统级备份。建议定期把重要笔记备份到独立位置。

## 变更

如果未来加入联网、账号、遥测或云同步功能，应在功能发布前更新本文件，并明确说明收集的数据、用途、保存期限和关闭方法。
