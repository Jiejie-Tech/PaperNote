# 隐私说明

最后更新：2026-07-21

PaperNote 是本地优先的笔记应用。

## 数据处理

- 不要求注册账号。
- 不包含广告、行为分析或遥测服务。
- Android 客户端不申请 INTERNET 权限。
- 笔记、墨迹、导入文件和设置默认只保存在用户设备上。
- 应用不会主动把笔记发送给项目维护者或第三方。

## 用户主动分享

当用户通过系统文件选择器、聊天工具、网盘或其他应用导出和分享 `.papernote`、PDF 或备份文件时，后续数据处理由用户选择的目标服务负责。请在分享前确认文件中不含不希望公开的信息。

## Android 数据删除

Android 资料库存放在应用私有目录。卸载应用、清除应用数据或恢复出厂设置可能永久删除本地笔记。执行这些操作前，请先把导出文件或整库备份保存到其他安全位置。

## 日志与问题报告

应用可能生成本地诊断信息。提交 Issue 时，请先检查截图、日志和样本文件，移除姓名、联系方式、文件路径和笔记内容等敏感信息。

## 联系

隐私或安全问题请通过仓库的 Issue 或 Security 渠道联系维护者。安全漏洞不应在公开 Issue 中附带可利用细节。
